Zapier + Claude API

Slack NL Parsing Flow

When users post meeting notes or deal updates to a dedicated Slack channel, Zapier triggers a Zap. Claude API parses the natural language text, Claude extracts structured data (account, type, content), then the result writes to the Notes sheet in Smartsheet and updates the parent Account row.

Smartsheet Automations

Automated Staleness Alerts

Smartsheet's built-in automation engine. When "Days Since Update" crosses a threshold, it sends alerts via Slack or email. No code required—just configure the rules in Smartsheet Automations.

Smartsheet
Automations: Pipeline Dashboard
5-Day Info Nudge

Then Send Slack notification to assigned owner
10-Day Warning
When Days Since Update = 10
Then Send Slack DM + flag row yellow
14-Day Escalation
When Days Since Update ≥ 14
Then Send Slack DM + notify manager
Monday Weekly Brief
When Scheduled every Monday 8:00 AM ET
Then Run summary report → post to #pipeline-updates
New Note → Slack Notification
When Row added to Notes sheet
Then Post summary to #deal-updates Slack channel
How it works: These automations run on Smartsheet's servers. You configure them once in the Automations UI—no code, no Zapier setup. Smartsheet checks conditions every time a row changes or on a schedule, then triggers actions (Slack messages, email, column updates, etc.). Perfect for sales ops workflows where time is critical.
